## 1. Alt Text on Visuals
| Visual | Alt Text Added | Description |
|--------|---------------|-------------|
| Big 6 Current % Card | ✅ | Shows average Big 6 dam level percentage |
| Big 6 Trend Line Chart | ✅ | Line chart showing dam levels from 2000 to 2026 |
| Current % by Dam Bar Chart | ✅ | Bar chart comparing current levels across all 14 dams |
| By Season Bar Chart | ✅ | Bar chart showing seasonal patterns in dam levels |

**How to add alt text in Power BI:**
- Right click visual → Edit alt text
- Add a clear description of what the visual shows

| Check | Status | Notes |
